# Conversión de Temperatura en Python

Este proyecto es un programa sencillo desarrollado en Python que permite convertir
una temperatura ingresada en grados Celsius a grados Fahrenheit.

## Funcionalidad
- Solicita al usuario una temperatura en grados Celsius.
- Convierte la temperatura a grados Fahrenheit.
- Indica si la temperatura está sobre el punto de congelación.

## Tecnologías utilizadas
- Python 3
- PyCharm
- GitHub
